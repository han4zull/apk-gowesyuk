# GowesYuk

Aplikasi penyewaan sepeda berbasis Laravel.  
Fitur:
- Pendaftaran & login user
- Penyewaan sepeda online
- Pembayaran & laporan

## Cara Install
1. Clone repo ini
2. Jalankan `composer install`
3. Copy `.env.example` ke `.env`
4. Jalankan `php artisan migrate`

## Lisensi
Bebas digunakan untuk belajar.
